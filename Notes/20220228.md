## 算法

> **二分查找**  
> 首先划分区间，满足条件与不满足条件  
> 然后寻找边界，是左边界还是右边界  
> 哪边界就check(mid)满足哪边条件  
> 模板  
>
> ```
> 确定右区间左边界
> int l = 0, r = size - 1;
> while (l < r) {
>   int mid = l + r >> 1;
>   if(check(mid)) r = mid;
>   else l = mid + 1;
> }
> return l;
> 确定左区间右边界
> int l = 0, r = size - 1;
> while (l < r) {
>   int mid = l + r + 1 >> 1;
>   if(check(mid)) l = mid;
>   else r = mid - 1; # mid-1前面就+1
> }
> return l;
> ```
>
> 查找出边界点后，要注意返回值的几种情况，对情况进行分类后输出结果。
